CLI do .NET
CLI - Command-Line Interface: ILC - Interface de Linha de Comando
	Processa os comandos que serão enviados para um programa
	de computador em forma de linhas de texto
Comandos do .NET
dotnet --version
dotnet --help: todos os comandos q pode usar
dotnet --info: 
dotnet new: retorna lista de tipos q pode criar
dotnet new --list
dotnet new --list --tag web: lista com filtro
dotnet new console -o myApp: cria uma pasta pra solução
dotnet run
dotnet build: compilação pra checar erros
dotnet restore: carrega as referências necessárias pro projeto
